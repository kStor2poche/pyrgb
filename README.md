## A simple command-line rgb color converter written in python 3

![Illustration of the command](./screenshot.png)
## Usage 
Use `rgb.py -h` to get help about its features ! 
It can :
- convert a color between 8 bit decimal, normalized decimals and hexadecimal formats
- display the color in neat little blocks if your terminal supports full color !
